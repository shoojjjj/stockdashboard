## 한 줄 정의

`오늘 매매할 것은?`이라는 요청은 개별 에이전트의 단독 결론이 아니라 토론 중재 에이전트의 회의 요청이다.

```text
자산제곱 = 원칙과 실행 가능성
소수몽키 = 시장 신호와 모멘텀
포트폴리오 = 실제 보유와 노출 여력
중재자 = 합의/불합의와 사용자 확인 문장
```
