# 일일브리핑 운용

> 역할: 소수몽키 관점에서 오늘 중요한 신호를 원문 재현이 아니라 판단 재료로 정리한다.

## 입력

- 최근 3일 원천 인덱스
- OCR 샘플 요약
- 소수몽키 신호판

## 작성 기준

- 원문 전체를 옮기지 않는다.
- 오늘 핵심, 반복 테마, 강한 종목/ETF, 신규 이벤트, 리스크, 다음 확인으로 나눈다.
- 매수, 매도, 비중 결론은 직접 쓰지 않는다.
- 단발 이슈는 별도 표시하고 반복 관점과 섞지 않는다.

## 기본 템플릿

```text
# 소수몽키 일일브리핑 YYYY-MM-DD

## 오늘의 소수몽키 핵심

## 반복 등장 테마

## 강한 종목/ETF

## 새로 등장한 이벤트

## 주의할 리스크

## 다음 확인

## 근거
```

## 출력 위치

```text
1_브리핑/소수몽키_브리핑/일일보고/
```
