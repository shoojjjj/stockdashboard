# 이미지 OCR 판독 운용

> 역할: Telegram 원본 사진에서 표, 히트맵, 티커, 수익률, 핵심 키워드를 샘플 추출한다.

## 입력

- 기본: `01_텔레그램_원천파서/API_수집/인덱스/소수몽키_API_원천인덱스.jsonl`
- 기본 원본 사진: `01_텔레그램_원천파서/API_수집/photos/*.jpg`
- 보조: `01_텔레그램_원천파서/인덱스/소수몽키_원천인덱스_전체.jsonl`
- 보조 원본 사진: `소수몽키/Telegram Desktop/ChatExport_2026-05-30/photos/*.jpg`

## 제외 규칙

- 파일명에 `_thumb`가 들어간 썸네일은 제외한다.
- Telegram 기본 이미지, 스티커, 비디오 파일은 제외한다.
- PDF는 OCR 대상이 아니다.

## OCR 기본값

```bash
tesseract image.jpg stdout -l kor+eng --psm 6
```

## 샘플링

- 최근 3일(`2026-05-28`, `2026-05-29`, `2026-05-30`) 원본 사진에서 날짜별 균형 샘플을 뽑는다.
- 첫 단계에서는 완전한 문장 복원이 아니라 아래 항목 추출률을 확인한다.

## 추출 항목

- 티커, ETF
- 종목명 또는 섹터명
- 상승률, 하락률, 퍼센트 값
- 실적, 가이던스, CAPEX, 금리, 유가, 관세, 정책, 수급 등 키워드
- 표, 히트맵, 단일 캡처 여부의 추정 분류

## 실행

```bash
python3 0_주식_에이전트/소수몽키_에이전트/02_이미지_OCR_판독/run_ocr_sample.py \
  --index "0_주식_에이전트/소수몽키_에이전트/01_텔레그램_원천파서/API_수집/인덱스/소수몽키_API_원천인덱스.jsonl" \
  --export-dir "0_주식_에이전트/소수몽키_에이전트/01_텔레그램_원천파서/API_수집" \
  --output-dir "0_주식_에이전트/소수몽키_에이전트/02_이미지_OCR_판독/OCR_샘플" \
  --from-date 2026-05-28 \
  --to-date 2026-05-30 \
  --max-images 12
```

## 출력

| 파일 | 설명 |
|------|------|
| `OCR_샘플_2026-05-28_2026-05-30.jsonl` | OCR 원문과 추출 키워드 |
| `OCR_샘플_2026-05-28_2026-05-30.md` | 검토용 OCR 샘플 요약 |

## 검증

- `_thumb` 파일이 OCR 결과에 없어야 한다.
- 티커, 종목명, ETF, 수익률, 핵심 키워드가 일부라도 추출되는지 확인한다.
- 표, 히트맵, 단일 캡처 분류가 과도하게 한쪽으로 몰리는지 확인한다.
