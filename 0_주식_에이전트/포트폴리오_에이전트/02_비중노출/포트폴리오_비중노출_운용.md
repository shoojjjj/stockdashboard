# 포트폴리오 비중노출 운용

> 역할: 실제 계좌가 어떤 테마와 위험에 노출되어 있는지 분류한다.

## 노출 축

| 축 | 예시 |
|----|------|
| 성장 코어 | `NVDA`, `TSLA`, `RKLB`, AI/우주 장기 가설 |
| AI 인프라 | 전력, 냉각, 광통신, 데이터센터, 서버, 메모리 |
| 실물 인프라 | 원전, 전력망, LNG, 정련, 환경처리 |
| 원자재/정책 | 금, 은, 구리, 우라늄, 희토류, 에너지 |
| 현금흐름/방어 | 단기채, 배당, 금융, 현금 |
| 실험/정찰병 | 고위험 소액 보유, 신규 테마 테스트 |

## 판단 규칙

- 같은 테마가 이미 많으면 좋은 후보도 `추가 금지`가 될 수 있다.
- 현금성 자산은 예수금과 단기채를 우선으로 본다.
- 장기채는 방어 자산이지 즉시 매수 총알로 보지 않는다.
- 주도주 수익이 커졌다면 추가매수보다 수익 보호를 먼저 본다.
- 빈 축을 채우는 후보는 같은 조건에서 우선순위가 높다.

## 토론회의 출력

토론회의에 넘길 때는 아래 형식으로 압축한다.

```text
포트폴리오 상태:
중복 노출:
현금/방어 여력:
추가 가능:
추가 금지:
수익 보호:
정리/감축:
```
