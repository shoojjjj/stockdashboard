새 날짜 문서를 계속 만들지 않는다. 실행 후에는 위 두 현재판을 덮어쓴다.

자산제곱 에이전트는 이 데이터를 입력으로 받아 원칙, 시나리오, 매수게이트에 맞는지만 판단한다.
