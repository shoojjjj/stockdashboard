# 포트폴리오 수익보호·정리 운용

> 역할: 고수익 보유의 보호 기준과 손실/역할 충돌 보유의 정리 우선순위를 관리한다.

## 수익 보호

수익이 큰 종목은 신규 후보가 아니라 보호 대상이다.

| 조건 | 처리 |
|------|------|
| 고수익이지만 장기 가설 유지 | 보유 유지, 추가매수 금지 |
| 고수익 + 변동성 확대 | 보호 기준 강화 |
| 스윙 계좌 고수익 | 일부 익절 또는 장기 이관 검토 |
| 장기 계좌 고수익 | 단기 손실 보전용 매도 금지 |

## 정리/감축

| 조건 | 처리 |
|------|------|
| 손실 확대 + 가설 부재 | `정리/감축 검토` |
| 역할 충돌 | 더 명확한 보유와 비교 후 감축 우선 |
| 손실 중 추가매수 욕구 | 새 근거 없으면 추가 금지 |
| 계좌 역할 불일치 | 장기/스윙 전환 근거 확인 전 보류 |

## 중재자에게 넘기는 말

포트폴리오 에이전트는 매수/매도 최종 결론을 내리지 않는다. 중재자에게 아래 네 가지 상태만 넘긴다.

```text
유지 가능:
추가 금지:
수익 보호:
정리/감축 검토:
```
